# 🖥️ FCFS CPU Scheduling (C Language)

This project implements the First Come First Serve (FCFS) CPU scheduling algorithm in C.

It takes Arrival Time and Burst Time as input and calculates:

- Completion Time (FT)
- Turnaround Time (TAT)
- Waiting Time (WT)
- Average Waiting Time
- Average Turnaround Time

---

## 🚀 Features

- Accepts random arrival times
- Handles CPU idle time
- Automatically sorts processes based on arrival time
- Beginner-friendly implementation in C

---

## ⚙️ How to Compile & Run

```bash
gcc fcfs.c -o fcfs
./fcfs
```

---

## 🧠 Algorithm Steps

1. Input number of processes  
2. Input Arrival Time and Burst Time  
3. Sort by Arrival Time  
4. Compute:
   - Finish Time
   - Turnaround Time
   - Waiting Time  
5. Display averages
